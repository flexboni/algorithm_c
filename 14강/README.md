# 스택(Stack)

스택(Stack)과 큐(Queue)는 컴퓨터 공학에서 가장 기본이 되는 자료구조이다. 쉬운 예로 **스택** 은 _택배 상하차_, **큐** 는 _은행 창구_ 라고 생각하면 쉽다.

    스택은 입구와 출구가 하나밖에 없는 상태
