# 큐(queue))

큐(Queue)는 컴퓨터 공학에서 가장 기본이 되는 자료구조 중 하나이다. _먼저 들어온 원소가 먼저 나가는 (First in First Out aka.FIFO) 특징이 있다._

    스택은 입구 하나와 출구가 하나가 있는 상태
